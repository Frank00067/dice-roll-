# 🎲 Dice Roll App

A Flutter app that rolls multiple dice at once with a single tap.

## Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install) installed
- A connected device, emulator, or browser

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Frank00067/dice-roll-.git
cd dice-roll-
```

### 2. Install dependencies

```bash
flutter pub get
```

### 3. Run the app

**On Chrome (recommended):**
```bash
flutter run -d chrome
```

**On Windows desktop:**
```bash
flutter run -d windows
```

**On a connected Android/iOS device:**
```bash
flutter run
```

## How to Use

- Launch the app
- Tap the **Roll Dice** button to randomly roll all 5 dice at once
- Each die shows a face from 1 to 6
